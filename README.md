This project is a simple full-stack weather application built as part of a UI Intern assessment. The goal was to consume the provided Weather API endpoints and design a clean, user-friendly interface to display the data.

The application allows users to:

View a list of Indian cities

Select a city from a dropdown

Fetch and display real-time weather details

Handle loading states and errors gracefully

To ensure security and avoid exposing API keys in the frontend, a backend proxy was implemented using Express.
